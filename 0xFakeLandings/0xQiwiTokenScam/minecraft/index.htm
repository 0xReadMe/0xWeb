Инструкция по получении API Token QIWI

В данном разделе Вы можете узнать как получить api token.

  • 1

    Небходимо перейти на сайт QIWI и авторизоваться на свой кошелек

    Переходим на сайт qiwi.com/api и заходим в свой кошелек.

  • 2

    Выпускаем новый токен.

    Нажимаете на кнопку выпустить новый токен.

  • 3

    Выпускаем полнофункциональный токен

    Выбираем все галочки и нажимаем и нажимаем продолжить.

  • 4

    Подтверждение по СМС и сам токен

    Теперь подтверждаем выпуск токена по СМС, и далее он нам показывается, копируем и вставляем сюда!
